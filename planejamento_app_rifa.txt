🎲 Estrutura e Lógica: Geração de Bilhetes Aleatórios
Vamos estruturar a lógica que o seu backend (Node.js/Next.js) precisa executar quando o Organizador finaliza a criação de uma rifa de 100 bilhetes divididos em 10 blocos:

1. Dados de Entrada
Rifa ID: (Ex: 5)

Total de Bilhetes/Números Únicos: N=100 (Números da sorte de 00 a 99).

Total de Blocos (Séries): B=10.

Bilhetes por Bloco (Série): T=N/B=100/10=10 cartões por bloco.

2. Algoritmo de Distribuição Aleatória
A chave é criar um pool de todos os números de sorte e distribuí-los uniformemente e aleatoriamente entre os blocos.

Criação do Pool: Crie um array de todos os 100 números de sorte (ex: ['00', '01', '02', ..., '99']).

Embaralhamento (Shuffle): Embaralhe (randomize) o array completo de 100 números. Isso garante a distribuição não sequencial.

Distribuição em Blocos:

Crie 10 "cestas" (representando os 10 blocos).

Itere sobre os 100 números embaralhados e coloque o primeiro número na cesta 1, o segundo na cesta 2, ..., o décimo na cesta 10, o décimo primeiro na cesta 1, e assim por diante. (Ou simplesmente, pegue os primeiros 10 para o Bloco 1, os próximos 10 para o Bloco 2, etc.)

3. Mapeamento no Banco de Dados
Para registrar essa distribuição, precisaremos de uma nova tabela que ligue o Número de Sorte ao seu Bloco e o status inicial.

🔑 Geração de QR Code com Next.js (API Route)
Para gerar a imagem do QR Code, criaremos uma API Route no Next.js. Isso é mais eficiente do que gerar o QR Code no frontend, pois permite que o código-fonte seja um buffer de imagem que pode ser facilmente inserido no HTML de impressão.

1. Instalação Necessária
No seu terminal, instale a biblioteca qrcode:

Bash
npm install qrcode
2. Criação da Rota da API (/pages/api/generate-qrcode.js)
Esta rota receberá a URL que você quer codificar e a devolverá como uma imagem PNG.

JavaScript
// /pages/api/generate-qrcode.js

import QRCode from 'qrcode';

export default async function handler(req, res) {
  // A URL que será codificada no QR Code (passada como query parameter 'url')
  const { url } = req.query;

  if (!url) {
    return res.status(400).json({ error: 'URL parameter is required.' });
  }

  try {
    // 1. Gerar o QR Code como um buffer PNG
    const qrCodeBuffer = await QRCode.toBuffer(url, {
      type: 'image/png',
      // Defina um nível de erro 'M' (Médio) para um bom equilíbrio entre legibilidade e densidade de dados
      errorCorrectionLevel: 'M', 
      scale: 8 // Ajusta o tamanho da imagem (pixels por módulo)
    });

    // 2. Configurar a resposta como uma imagem
    res.setHeader('Content-Type', 'image/png');
    res.setHeader('Cache-Control', 'public, max-age=31536000, immutable'); // Cache de 1 ano
    
    // 3. Enviar o buffer da imagem
    res.send(qrCodeBuffer);
    
  } catch (error) {
    console.error('QR Code Generation Error:', error);
    res.status(500).json({ error: 'Failed to generate QR code.' });
  }
}

2. 🧠 A Lógica Principal: encontrarGanhadorPorProximidade
Esta função é o coração do seu sistema e precisa implementar sua regra: procurar o número posterior e anterior (que foi PAGO) até encontrar o ganhador.

Para um sorteio de 100 bilhetes (00-99), a lógica seria:

JavaScript
// Função simplificada (deve ser implementada no seu backend Node.js)
async function encontrarGanhadorPorProximidade(rifaId, numeroSorteado) {
    const maxNum = (await getRifa(rifaId)).qtde_bilhetes - 1; // Ex: 99 ou 9999
    const num = parseInt(numeroSorteado, 10);
    
    let offset = 0;
    while (offset <= maxNum) { // Evita loop infinito
        
        // 1. Verificar Número Posterior
        const posterior = (num + offset) % (maxNum + 1); // Garante que 99 + 1 = 00
        const ganhadorPosterior = await buscarBilhetePago(rifaId, formatNumber(posterior, maxNum));

        if (ganhadorPosterior) {
            return {
                numeroSorteadoOriginal: formatNumber(num, maxNum),
                numeroGanhador: formatNumber(posterior, maxNum),
                comprador: ganhadorPosterior.comprador_nome,
                motivo: offset === 0 ? 'Exato' : 'Posterior mais próximo'
            };
        }

        // 2. Verificar Número Anterior (Se offset > 0)
        if (offset > 0) {
            const anterior = (num - offset + maxNum + 1) % (maxNum + 1); // Garante que 00 - 1 = 99
            const ganhadorAnterior = await buscarBilhetePago(rifaId, formatNumber(anterior, maxNum));
            
            if (ganhadorAnterior) {
                return {
                    numeroSorteadoOriginal: formatNumber(num, maxNum),
                    numeroGanhador: formatNumber(anterior, maxNum),
                    comprador: ganhadorAnterior.comprador_nome,
                    motivo: 'Anterior mais próximo'
                };
            }
        }
        
        offset++;
    }
    return null; // Caso improvável de não haver bilhetes pagos
}

// Helper para formatar o número com zeros à esquerda (Ex: 5 -> '05')
function formatNumber(n, max) {
    const padding = (max + 1).toString().length - 1;
    return String(n).padStart(padding, '0');
}
