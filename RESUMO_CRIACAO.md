- Prioridade: posterior → anterior
- Apenas bilhetes PAGOS

- Mesma interface de funções
- Dados realistas pré-configurados

### 3. Geração Aleatória Justa

Algoritmo Fisher-Yates:
- Embaralhamento perfeito
- Distribuição uniforme nos blocos
- Sem repetições

- QR do Vendedor: Registro
- QR do Comprador: Validação
- Geração dinâmica via API
